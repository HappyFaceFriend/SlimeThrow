# SlimeThrow


## 개발 규칙
### Unity 설정
- 유니티 버전 : 2021.3.8f1

### Unity 개발 규칙
- 자기 씬에서만 작업하기
  - 다른 사람이 만든 Prefab은 물어보고 변경하기 or 변경 후 말해주기 or 다른 브랜치에서 작업하기
- 다른 사람의 코드나 여러 씬을 한번에 수정하는 리팩토링 할 땐 꼭 브랜치 새로 파서 하기
- 이미지 Import 시
  - Pixels Per Unit : 32
  - Filter Mode : Point
  - Compression : None
